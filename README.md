# Multiplayer Snakes & Ladders

## About the Game

A remake of the classic Snakes and Ladders Board game.

### Objectives.

- Learning the basics of socket programming using **Node.js** and **socket.io**

## Tech stack Used

- HTML / CSS / Javascript
- Node.js
- socket.io
